BookMyShow - Get a notification when the movie tickets are available

Nov 18, 2018 - 1 minute read

This python script checks if movie tickets are available for a showtime and send an email notification. I’ve written this in Python 3 using BeautifulSoup.

Run the script using python3 app.py.

Please feel free to fork and file issues or pull requests. https://github.com/saikiransripada/bookmyshow
